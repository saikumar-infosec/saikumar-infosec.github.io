Critical Severity

📌 Server-Side Template Injection (SSTI)

🧠 Description

What is SSTI?

Server-Side Template Injection occurs when user input is unsafely embedded into a template engine. Attackers can inject template directives to execute arbitrary code on the server.

Common Template Engines: Jinja2 (Python), Twig (PHP), FreeMarker (Java), Velocity (Java), ERB (Ruby), Handlebars (Node.js), Pug (Node.js)

🏷️ Classification

  • Type: Injection
  • CWE: CWE-94: Improper Control of Generation of Code

⚙️ Exploitation Steps

  1. Identify template engine by injecting math expressions
  2. Confirm vulnerability when math is evaluated
  3. Use engine-specific payloads to read files
  4. Escalate to RCE using built-in functions

🛡️ Mitigation

✅ Never trust user input in templates
✅ Use sandboxed template engines
✅ Escape user input before embedding
✅ Limit template functionality
✅ Regular security audits
