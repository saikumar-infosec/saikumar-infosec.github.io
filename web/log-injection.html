Low Severity

📌 Log Injection (Log Forging)

🧠 Description

Attackers inject newline characters or malicious data into log entries, forging fake log lines, hiding their actions, or exploiting log viewers (e.g., JavaScript injection in log viewers).

💣 Basic Payload

User-Agent: \n2023-01-01 Admin logged in from 127.0.0.1\n

🛡️ Mitigation

✅ Sanitize/encode log input (remove CR/LF).
✅ Use structured logging (JSON) to separate fields.
✅ Never render raw logs in a browser without escaping.
