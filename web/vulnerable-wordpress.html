Critical Severity

📌 Vulnerable WordPress Plugins & Themes

🧠 Description

WordPress sites using outdated or poorly coded plugins/themes expose SQLi, XSS, RCE, and auth bypass vulnerabilities.

🔍 Detection

wpscan --url https://target.com --enumerate vp,vt

💣 Example: Vulnerable Plugin RCE

https://target.com/wp-content/plugins/vulnerable-plugin/ajax.php?cmd=id

🛡️ Mitigation

✅ Keep WordPress core, plugins, themes updated.
✅ Remove unused plugins/themes.
✅ Use a security plugin (Wordfence, Sucuri).
✅ Regularly run wpscan.
