📌 XXE (XML External Entity) Injection

🧠 Description

What is XXE?

XML External Entity (XXE) injection is a vulnerability that allows an attacker to interfere with an application's processing of XML data. It occurs when XML input containing a reference to an external entity is processed by a weakly configured XML parser.

Types of XXE Attacks:

  • File Disclosure: Read local files using file:// protocol
  • SSRF via XXE: Make the server send requests to internal systems
  • Blind XXE: Exfiltrate data via out-of-band techniques
  • Denial of Service (Billion Laughs): Cause memory exhaustion
  • XInclude Attacks: When full DTD parsing is disabled

Security principle violated: Secure Configuration / Input Validation

🏷️ Classification

  • Type: XML External Entity Injection
  • OWASP: A05:2021 – Security Misconfiguration
  • CWE: CWE-611: Improper Restriction of XML External Entity Reference

🎯 Attack Surface

  • ✅ SOAP APIs
  • ✅ XML-RPC endpoints
  • ✅ File uploads (.xml, .svg, .docx, .xlsx)
  • ✅ REST APIs that accept XML
  • ✅ SAML assertions
  • ✅ RSS/Atom feeds
  • ✅ Office document metadata

💣 Basic Detection Payloads

<!DOCTYPE root [<!ENTITY test "TEST">]><root>&test;</root>

⚙️ Exploitation Steps

  1. Identify XML input (SOAP, file upload, API)
  2. Inject basic DOCTYPE with external entity
  3. Test file disclosure with /etc/passwd
  4. If no output, try blind XXE with OOB
  5. Escalate to SSRF to access internal services
  6. Use XXE to read source code via PHP wrappers

🛡️ Mitigation

✅ Disable External Entities: Configure XML parsers to disable DTDs
✅ Use Secure Parsers: Use JSON where possible, or secure XML parsers
✅ Input Validation: Validate XML structure before parsing
✅ Upgrade Libraries: Keep XML libraries updated
✅ Disable DOCTYPE: Configure parser to reject DOCTYPE declarations 
# Java - Disable XXE
DocumentBuilderFactory dbf = DocumentBuilderFactory.newInstance();
dbf.setFeature("http://xml.org/sax/features/external-general-entities", false);
dbf.setFeature("http://xml.org/sax/features/external-parameter-entities", false);

# Python - defusedxml
from defusedxml import minidom
doc = minidom.parseString(xml_string)

# PHP
libxml_disable_entity_loader(true);

⚠️ Risk / Impact

Severity: Critical

Impact:
  • Read arbitrary files (/etc/passwd, configuration files, source code)
  • SSRF to internal networks
  • Denial of Service via entity expansion
  • Port scanning internal infrastructure
  • Potential RCE via expect:// or PHP wrappers
