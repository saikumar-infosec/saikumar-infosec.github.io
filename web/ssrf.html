📌 Server-Side Request Forgery (SSRF)

🧠 Description

What is SSRF?

Server-Side Request Forgery (SSRF) is a vulnerability that allows an attacker to induce the server to make requests to an arbitrary domain, including internal services that are not directly accessible from the internet.

Types of SSRF:

  • Basic SSRF: Response is returned to the attacker
  • Blind SSRF: No response, but requests are made (detect via DNS/HTTP callbacks)
  • Partial SSRF: Only partial response or error messages

Security principle violated: Input Validation / Network Segmentation

🏷️ Classification

  • Type: Server-Side Request Forgery
  • OWASP: A10:2021 – Server-Side Request Forgery
  • CWE: CWE-918: Server-Side Request Forgery (SSRF)

🎯 Attack Surface

  • ✅ URL parameters (?url=, ?callback=, ?dest=)
  • ✅ POST body (JSON with URL fields)
  • ✅ File uploads (URL in filename or metadata)
  • ✅ API endpoints that fetch external resources
  • ✅ Webhooks
  • ✅ Image/proxy endpoints
  • ✅ PDF generators

⚙️ Exploitation Steps

  1. Identify parameter that fetches URLs (avatar, webhook, proxy)
  2. Test with http://127.0.0.1:22 to check response
  3. Use Burp Collaborator or Interactsh for blind detection
  4. Enumerate internal ports (port scanning via status codes/timeouts)
  5. Access cloud metadata endpoints
  6. Use gopher/dict protocols to interact with internal services
  7. Escalate to RCE via Redis/MySQL/other services

🛡️ Mitigation

✅ Allowlist Approach: Only allow requests to trusted domains
✅ Deny Internal IPs: Block access to private IP ranges
✅ Disable Unnecessary Protocols: Disable file://, dict://, gopher://
✅ Network Segmentation: Isolate internal services
✅ Response Handling: Don't return raw responses to client
✅ Regular Audits: Scan for SSRF vulnerabilities

⚠️ Risk / Impact

Severity: Critical

Impact:
  • Access cloud metadata (AWS IAM credentials)
  • Port scan internal network
  • Access internal services (Redis, MySQL, Kubernetes API)
  • Read local files (file:// protocol)
  • RCE via Redis/MySQL/other services
